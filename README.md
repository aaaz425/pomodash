# Pomodash

포모도로 타이머 기반 학습 집중 도구 — 작업 계획, 집중 타이머, 세션 기록, 통계 대시보드를 하나에서

## Features

- **작업 관리** — 오늘 할 작업 목록 생성 및 카테고리 태그
- **포모도로 타이머** — 집중/휴식 사이클 반복, 브라우저 알림 + 소리 알람
- **동기부여 메시지** — 세션 중 랜덤 명언 및 응원 메시지
- **세션 기록** — 세션 종료 후 배운 것, 느낀 것 메모
- **대시보드** — 일/주/월별 공부 시간 차트 및 스트릭

## Tech Stack

Next.js 16 · Tailwind CSS · shadcn/ui · Zustand · Recharts · Vercel

## Getting Started

```bash
npm install
npm run dev
```
