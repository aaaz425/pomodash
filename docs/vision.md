# Product Vision

## 왜 만드는가

현대인의 집중력은 점점 짧아지고 있다. SNS 알림, 유튜브 추천 영상, 끊임없는 메시지들이 주의를 빼앗는다.
그러나 수험생은 공부해야 하고, 취업 준비생은 스펙을 쌓아야 하며, 직장인은 자기계발을 멈출 수 없다.

**Pomodash는 "해야 할 일이 있는 사람"이 실제로 그 일을 해낼 수 있도록 돕는다.**

포모도로 기법(집중-휴식 반복 사이클)을 기반으로, 작업 계획 → 집중 → 기록의 흐름을 하나의 앱에서 완성한다.
동기부여 메시지와 누적 기록이 "오늘도 해냈다"는 성취감으로 이어지도록 설계한다.

## 타겟 유저

- 수험생 (수능, 공무원, 자격증 등)
- 취업 준비생
- 자기계발이 필요한 직장인
- 집중이 어려운 모든 사람

## 디자인 방향

- **다크 모드 우선** — 밤에 공부하는 유저가 많음
- **미니멀** — 타이머가 화면의 중심, 방해 요소 최소화
- **집중 모드에서 다른 UI 숨김** — 집중 모드 진입 시 타이머와 동기부여 메시지만 노출
- 컬러: 딥 네이비 + 포인트 민트/그린 계열 (집중/침착 느낌)
