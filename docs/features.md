# Features

확정된 기능 명세. 기획 논의 + 기존 docs/ 문서 기준으로 정리.

---

## 1. 타이머

### 기본 동작
- 집중 / 짧은 휴식 / 긴 휴식 3단계 사이클
- 기본값: 집중 25분 / 짧은 휴식 5분 / 긴 휴식 15분 / 4사이클 후 긴 휴식
- 시작 / 일시정지 / 리셋
- 타이머 종료 시 강제 전환 없음 — 소리 알람 + 브라우저 알림만, 이후는 사용자 선택

### 타이머 철학
보조 도구. "몇 분 집중했나"를 보여주는 참고 수단이지, 지켜야 하는 규칙이 아님.
세션 도중 언제든 종료 가능하며 실제 집중한 시간만 기록됨.

### 세션별 설정
- 세션 시작 전 집중 / 짧은 휴식 / 긴 휴식 / 사이클 수 설정 가능
- 전역 설정값이 기본으로 채워짐

### 집중 모드 (MVP 포함)
- 타이머 실행 중 진입 가능
- 화면을 어두운 오버레이로 덮고 타이머 링 + 동기부여 메시지만 노출
- 버튼으로 종료

### 페이지 이동 시
- 타이머는 계속 동작 (절대 시간 기반 — 백그라운드 탭 스로틀 무관)
- 다른 페이지에 있을 때 nav에 미니 타이머 표시 → 클릭 시 메인으로 이동

### 작업 전환
- 타이머 도중 다른 작업으로 전환 가능 (타이머 유지)

---

## 2. 작업 관리

- 작업 생성 / 수정 / 완료 / 삭제 (CRUD)
- 카테고리 태그 (기본: 공부 / 업무 / 운동 / 독서)
- 목표 시간 설정 (`targetMinutes`) — 작업 진행도 표시용
- 드래그앤드롭으로 순서 조정
- 장기 계획과 연결 가능
- 서브태스크 (작업 내 체크리스트) — MVP 이후

---

## 3. 장기 계획

- 월 / 분기 단위 목표 설정
- 작업 생성 시 연결할 장기 계획 선택 가능
- 대시보드에서 계획별 진척도 확인

---

## 4. 동기부여 메시지

- 집중 모드에서만 노출
- 하드코딩된 기본 명언 목록 (랜덤 노출)
- 사용자가 직접 메시지 추가 / 삭제 가능 (설정에서)

---

## 5. 세션 기록

- 세션 종료 후 메모 작성 (배운 것, 느낀 것 등)
- 저널 화면(`/journal`)에서 히스토리 조회
- 기록 수정 / 삭제 가능

---

## 6. 대시보드

- 일 / 주 / 월 탭 필터
- 오늘 집중 시간 + 목표 시간 설정 및 달성률
- 연속 기록 (스트릭)
- 카테고리별 색상 구분 막대 차트 (Recharts)
- 장기 계획 진척도

---

## 7. 설정

- 타이머 기본값 설정 (집중 / 짧은 휴식 / 긴 휴식 / 사이클 수)
  - 허용 범위: 집중 1–60분 / 짧은 휴식 1–30분 / 긴 휴식 1–60분 / 사이클 1–10회
- 소리 알람 on/off
- 브라우저 알림 on/off
- 동기부여 메시지 관리 (추가 / 삭제)
- 카테고리 관리 (추가 / 수정 / 삭제)

---

## 8. 공통

- 로그인 없음 (localStorage 기반, MVP)
- 다크 모드 우선 + 라이트 모드 지원 (시스템 설정 연동, 수동 전환 가능)
- PWA — 모바일 홈 화면 추가 가능
- Vercel 배포

---

## MVP 이후 (백로그)

- 서브태스크 (작업 내 체크리스트)
- Supabase + Clerk (클라우드 동기화 + 로그인)
